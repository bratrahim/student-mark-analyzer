In order to compile the source code, you have to create a project
from the .java files provided in the source code folder. Do not forget
to attach libraries provided in the libs folder.

To start the program double click on Analyzer.jar.

Classpath of the software is 
java -jar Analyzer.jar

System requirements:
-JRE Java Runtime Environment 1.8 installed on Mac OS X from 10.8.3+ version,
Windows from 7, Linux from 5.5+.
